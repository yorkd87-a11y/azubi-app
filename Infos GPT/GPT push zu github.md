# GPT Push zu GitHub

Nutze fuer neue Auftraege einfach diese Vorlage:

```text
Bitte aendere in der App: [deine Aenderung].
Danach alles committen und auf GitHub pushen.
```

Wenn nichts geaendert werden muss und ich nur hochladen soll:

```text
Bitte alles committen und auf GitHub pushen.
```

Wenn ich erst pruefen und dann hochladen soll:

```text
Bitte pruefe die letzten Aenderungen, behebe gefundene Probleme und pushe danach auf GitHub.
```
